시험 구조
실전 모의고사 (50문항 / 60분)
1~17번: 손해보험 (17문항)
18~33번: 공통 (16문항)
34~50번: 제3보험 (17문항)
실제 손해보험 자격시험과 동일한 구조로 랜덤 출제됩니다.
과목별 연습 (20문항 / 25분)
선택한 과목의 문제 풀에서 20문항을 랜덤 출제합니다.
오답 재시험
직전 시험에서 틀린 문항만 다시 풀 수 있습니다.
채점 방식
실전 모의고사 — 과목별 채점
손해보험 점수 = (손해 1~17번 + 공통 18~33번) 정답 / 33 x 100
제3보험 점수 = (공통 18~33번 + 제3보험 34~50번) 정답 / 33 x 100
합격 기준: 각 과목 60점 이상
과목별 연습 — 단순 정답률
전체 정답 수 / 전체 문항 수 x 100
문제 데이터
총 241문항 (손해보험 92 / 공통 86 / 제3보험 63)
출처: 25년 모의고사 #1~#3 + 26년 예상문제 전체범위
중복 9문항 제거, 과목별 분류 완료
Gemini 해설
오답 문제에 대해 AI 해설을 제공합니다.
Google AI Studio에서 API 키를 발급받아 위 입력란에 입력하세요.
키는 이 기기의 브라우저에만 저장되며 외부로 전송되지 않습니다.
문제와 관련된 참고자료만 추출해서 보내는 RAG 방식으로 동작합니다.
데이터 저장
모든 데이터는 브라우저 localStorage에 저장됩니다.
시험 기록은 최대 30개까지 보관됩니다.
"전체 삭제" 버튼으로 기록을 초기화할 수 있습니다.
다른 기기나 브라우저와 동기화되지 않습니다.
