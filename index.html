시험 구조
실전 모의고사 (50문항 / 60분)
1~17번: 손해보험 (17문항)
18~33번: 공통 (16문항)
34~50번: 제3보험 (17문항)
실제 손해보험 자격시험과 동일한 구조로 랜덤 출제됩니다.
과목별 연습 (20문항 / 25분)
선택한 과목의 문제 풀에서 20문항을 랜덤 출제합니다.
오답 재시험
직전 시험에서 틀린 문항만 다시 풀 수 있습니다.
채점 방식
실전 모의고사 — 과목별 채점
손해보험 점수 = (손해 1~17번 + 공통 18~33번) 정답 / 33 x 100
제3보험 점수 = (공통 18~33번 + 제3보험 34~50번) 정답 / 33 x 100
합격 기준: 각 과목 60점 이상
과목별 연습 — 단순 정답률
전체 정답 수 / 전체 문항 수 x 100
문제 데이터

생명보험: 회차 단위 출제 (2025년판 교재 기준 모의고사 회차)
연도/회차를 선택해 실제 시험과 동일한 40문항 50분 형식으로 풀이
